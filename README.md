## console-app-node

Esta es una aplicación de consola y a su vez también de portafolio desarrollada en node v15.7.0 la cual integra entre otras cosas el modulo de inquirer para desarrollar menus interactivos . termina siendo un ABM pero con algunas features interesantes en su desarrollo.
